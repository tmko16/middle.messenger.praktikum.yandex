### Команды 
-----
- Сборка проекта 

```
npm run build
```
-------------
- Запуск dev сервера для разработки 

```
npm run dev
```
-------------------
- Запуск локального сервера 

```
npm run start
```

Запускает сборку проекта, а также поднимает express сервер на 3000 порту.
