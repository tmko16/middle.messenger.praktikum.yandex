### Команды 
-----
- Сборка проекта 

```
yarn run build
```
-------------
- Запуск dev сервера для разработки 

```
yarn run dev
```
-------------------
- Запуск локального сервера 

```
yarn run start
```

-------------------
- Запуск тестов

```
yarn run test
```

-------------------
- Запуск Линтера 

```
yarn run lint
```
